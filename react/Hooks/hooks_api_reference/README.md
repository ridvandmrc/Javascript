### Hooks API
* React has several predefined hooks
* It can be grouped like Basic, Additional and Library

### Basic Hooks
* useState
* useEffect
* useContext
# Additional Hooks
* useReducer
* useCallback
* useMemo
* useRef
* useImperativeHandle
* useLayoutEffect
* useDebugValue
* useDeferredValue
* useTransition
* useId
### Library Hooks
* useSyncExternalStore
* useInsertionEffect
